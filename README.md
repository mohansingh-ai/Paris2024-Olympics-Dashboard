# 🏅 Paris 2024 Olympics Dashboard

An interactive and visually rich Power BI dashboard analyzing the **Paris 2024 Olympic Games** — covering athletes, medal counts, gender distribution, country performance, and historical trends.

## 📊 Dashboard Sections

### 🧩 Overview
- **Total Athletes**: 11,113 (5,658 Male | 5,455 Female)
- **Participating Countries**: 206
- **Teams**: 1,698
- **Medals Summary**: 🥇22 | 🥈22 | 🥉23
- **Top Performing Country**: United States
- **World Map** with bubble indicators for country-wise medals

### 👟 Athletes Section
- Age & gender distribution
- Gender-wise medal breakdown
- Country vs. gender medal insights

### 🕰️ Historical Section
- Medal trends from 1896–1912
- Country-wise medal stats over time
- Includes Soviet Union, Germany, Australia, and more

## 🛠️ Tools Used
- **Microsoft Power BI**
- Custom **DAX Measures**
- **Map Visuals**, **Donut Charts**, **Bar Graphs**
